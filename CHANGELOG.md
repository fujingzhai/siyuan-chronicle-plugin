# Changelog

## 1.2.0 - 2026-07-17

- 新增临时面板锁：年份栏设置按钮右侧增加锁按钮，快捷键 L 可一键锁定或解锁；锁定时岁时记内容完全遮蔽、不可交互，解锁后保留原视图与位置。
- 日期面板改为不受事件数影响的响应式活动轨道；多条活动在日期下方等距展开，每条横线具有独立点击槽，更多活动才自动压缩。
- 收紧月份上下留白、月间距和全年视图底部空间；月历剩余高度在首尾与日期行之间均匀分配，并为末行保留完整活动轨道。

## 1.1.0 - 2026-07-16

- 新增日期面板：日历式年视图，12 张月历卡排开全年，有活动的日子显示类目色圆点，月历下方平铺当月活动卡片。
- 日期面板支持记录发生在某一天或几天的活动（点击日期格即记录），与主面板共享类目并同样支持绑定笔记。
- 时间视图中填了具体日期的活动会自动出现在日期面板；日期面板专属活动不干扰时间视图。
- 左侧年份栏两种视图共用；快捷键 D/W 切换视图，底部按钮顺序调整为「设置｜←｜→｜切换」。

## 1.0.2 - 2026-07-16

- 首次打开岁时记面板时，周时间轴会在布局完成后自动定位到本周。
- 切换年份后返回当前年份时，周时间轴会重新定位到本周。

## 1.0.1 - 2026-07-15

- 修复既有活动新建、选择或解绑笔记后，绑定关系未即时保存的问题。
- 确保绑定笔记后面板立即显示文档图标，并可点击跳转至对应思源文档。

## 1.0.0 - 2026-07-14

- 完成以年、季、月、周为层级的单一时间视图。
- 支持活动创建、编辑、删除、具体日期、类目、备注与多笔记绑定。
- 支持同时间格排序及跨时间格拖动。
- 支持时间笔记自动创建、默认笔记本安全迁移及文档生命周期核对。
- 支持面板内快捷键、可调列宽、类目拖动排序与颜色选择。
- 加入串行持久化、上一版本地备份、双语 README 与集市发布工程。
