# 岁时记（Chronicle）

简体中文 | [English](./README.md)

岁时记是一款按年、季、月、周组织活动的思源笔记插件。

一项活动可以属于某一整年、某个季度、某个月或某一周，同时再记录实际发生的具体日期。例如，一次旅行可以归入“2026 年第 3 季度”，并注明实际日期为“7 月 12 日至 7 月 18 日”。岁时记把不同时间尺度的活动集中在同一个年度面板中，用于记录、排序和回看；需要详细内容时，活动还可以连接思源中的原生笔记。

## 功能

- 同屏显示年、季、月、周四个时间尺度；周按完整周序列滚动浏览。
- 使用左右按钮或方向键连续切换年份，按 `T` 定位到今天所在的年、月和周。
- 活动包含标题、类目、所属时间、具体日期、备注和绑定笔记。
- 年、季活动逐项排列；月、周活动紧凑显示。各列宽度可拖动调整并自动保存。
- 活动可在同一时间格内拖动排序，也可拖到另一个时间格；所属时间和具体日期会随之调整。
- 点击年、季、月、周标题，可按层级创建或打开对应的时间笔记。
- 每个活动可搜索并绑定多篇既有笔记，也可按活动标题新建笔记并绑定。
- 类目支持新增、改名、选择颜色、拖动排序和删除。

## 笔记与笔记本

首次使用时，岁时记会使用或创建名为“岁时记”的笔记本。默认笔记本以思源笔记本 ID 识别，因此修改笔记本名称不会影响使用。

时间笔记按“年 → 季 → 月 → 周”建立层级。通过活动创建的笔记按“类目 → 活动标题”建立。更换默认笔记本时，只迁移这两类由岁时记创建和管理的笔记；从其他位置绑定的既有笔记不会被移动。

活动与绑定笔记彼此独立：移动或重命名笔记不会破坏绑定；删除笔记只会解除绑定，不会删除活动。删除活动时，可自行选择是否同时删除绑定笔记。

## 快捷键

- `T`：定位今天
- `N`：创建活动
- `S`：打开设置
- `←` / `→`：切换年份

快捷键仅在岁时记面板可见且没有进行输入或编辑时生效。
