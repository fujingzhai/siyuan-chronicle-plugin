# 岁时记（Chronicle）

简体中文 | [English](./README.md)

日历善于安排未来，岁时记更关心时间如何被真正度过。

岁时记取意《荆楚岁时记》，把一年展开为年、季、月、周四种尺度，让旅行、阅读、研究、节令与日常片段各归其时，并在需要时回到背后的笔记。它不是待办清单，而是一份可以不断积累、随时回看的个人编年记录。

## 核心功能

- 在同一面板中纵览年、季、月、周，快速切换年份或定位今天。
- 记录活动标题、所属时间、类目、具体日期与备注。
- 拖动调整活动顺序，也可直接拖到另一时间范围。
- 点击时间标题即可创建或打开对应的思源时间笔记。
- 为活动绑定一篇或多篇既有笔记，也可新建笔记并绑定。
- 自定义类目名称、顺序与颜色，建立自己的记录体系。

## 与思源笔记协同

活动与笔记彼此独立：不绑定笔记也能完整记录；删除绑定笔记只会解除关联，不会删除活动。已绑定的笔记即使移动位置仍可正常打开，修改标题后也会自动同步。

岁时记首次使用时会使用或创建同名笔记本。更换默认笔记本时，只迁移岁时记创建的时间笔记和活动笔记，不会移动你从其他位置绑定的既有文档。

## 快捷键

- `T`：定位今天
- `N`：创建活动
- `S`：打开设置
- `←` / `→`：切换年份

快捷键只在岁时记面板中生效，输入和编辑活动时不会误触。
